I - Installing Star Wars Galaxies


Skip to Part II if you already have Star Wars Galaxies installed.

You will need a copy of Star Wars Galaxies. You DO NOT need to have any of the expansions, and not having the expansions will not affect your game-play at all. The game can be purchased from Amazon and Ebay for a range of different prices. You can use anything that contains Star Wars Galaxies: An Empire Divided. Had you previously purchased the game through Steam, please continue reading on how to re-download it if you uninstalled it. If not, then you can skip the Steam section below. Steam

  1. Click on Library and select ALL GAMES.
  2. Look for the greyed out “Star Wars Galaxies” and click on it.
  3. Click on install.
  4. In the new window that comes up, take note of the install location for the game (the text inside the dropdown menu). In my case, it will be installed under C:\Program Files (x86)\Steam. Once you have done that, press Next.
  5. A window will now ask if you want to have a shortcut placed on your desktop. There is no point for this unless you plan on trying to connect to SoE's servers, which you can't do anymore.
  6. Now the game will start installing. Ignore the next window that comes up with the broken link and press Finish.
  7. Once it has finished installing, you can continue on to Part II.

Non-Steam

  1. Insert the game disc inside the CD Drive.
  2. Click Next on the first welcome screen.
  3. Click Next again on the welcome screen.
  4. Read the License Agreement fully and select I Agree.
  5. On the next screen, take note of the Destination Folder. You will need to know this to install SWG Legends.
  6. Select Next after that.
  7. Let the game install, inserting the discs it asks for when necessary. Read all the fine print inside your old game manual while you wait and admire the screenshots.
  8. Once it has finished installing, you can continue on to the next part.

II - Updating SWG Leviathan


Download the launcher files here:

Zip Method: SWGLevaiathanLauncher.zip.

Open the zip file linked above and extract the contents of it to the folder where SWG is installed (the default location is C:\Program Files (x86)\StarWarsGalaxies.) From that folder right click the SWGLevaiathanLauncher.exe file and run it as Administrator. Login to the launcher using your forum credentials. Note: Do not use the same folder for multiple emulators/projects. We have our own patches and files.

Follow the guide here for using our game launcher.

If you don't already have an account on our forums, make one here. This account information will be used to log into the game, as well.

You'll see it scanning your game directory and checking for updates. If an update is required, you will need to press the “Update” button to download the needed patches before you can play. Make sure SWG isn't running when you update.

Once you are fully patched, you can press “Play” to launch SWG.

If you have any trouble getting connected, speak to someone from CSR Staff in Discord or submit a ticket with Account Support Services.
